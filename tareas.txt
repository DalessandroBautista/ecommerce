Para completar el backend necesitamos:
Integración con sistema de pagos (Stripe/MercadoPago)
Autenticación y autorización (JWT)
Manejo de imágenes de productos (upload a S3 o similar)
Sistema de emails (confirmaciones, recuperación de contraseña)
Para el frontend necesitamos:
Páginas principales:
Listado de productos
Detalle de producto
Carrito
Checkout
Órdenes
Perfil de usuario
Integración con React Query
Manejo de estado del carrito
Formularios de checkout
Integración con pasarela de pagos
Infraestructura:
Configuración de Docker para producción
Configuración de base de datos en producción
Sistema de caché (Redis)
CDN para imágenes
Monitoreo y logs
